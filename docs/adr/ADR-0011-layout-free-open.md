# ADR-0011: レイアウト「大きく→コメント」のFree開放

## Status

Accepted

## Context

写真ごとのキャプション機能（#276）を追加するにあたり、PDFレイアウト「大きく（1ページ1枚）」を「コメント」に改称し、写真＋キャプションをセットで表示するモードとする。

現状「大きく」はPro限定出力（Freeはプレビューのみ）だが、Free版ユーザーがキャプションを入力しても PDF に反映されないという矛盾した UX が生まれるため、Free版でも出力可能にする。

## Decision

- PDFレイアウト「大きく」を「コメント」に改称し、Free版でも出力可能にする
- constraints.md の Free/Pro 不変差分からレイアウト行を削除する
- 課金画面の機能比較テーブルからレイアウト行を削除する
- 設定画面の Pro 特典リストからレイアウト項目を削除する
- RevenueCat の entitlement 設定は変更しない（アプリコード側のみ変更）

## Drivers

- **UX 一貫性**: 入力→出力の一貫性を確保する（入力できるのに出力に反映されない機能を作らない）
- **収益影響の限定性**: レイアウト選択は Pro 転換の主要ドライバーではない（月5回制限・写真枚数制限が主要）
- **コード品質**: Pro ゲートロジック削除によるシンプル化

## Alternatives

1. **「コメント」モードもPro限定に維持** → 却下: Free版でキャプション入力→PDF未反映が悪体験
2. **「標準」モードにもキャプション表示** → 却下: 2枚/ページのスペースにキャプション表示は窮屈。追加工数に見合わない
3. **3モード構成（標準/大きく/コメント）** → 却下: UIの複雑さ・テストケース増大に見合う価値がない

## Consequences

### 良い影響

- Free版ユーザーの PDF 品質向上（キャプション付き出力が可能に）
- PDF プレビュー画面の Pro 判定ロジックが 1 つ減りシンプルになる
- テストケースが削減される（Free+大きく→Proモーダルのテストが不要に）

### リスク

- Pro 契約者が価値減少を感じる可能性 → 対策: リリースノートで残り4特典の価値を強調
- ストア掲載の Pro 説明文の更新が必要 → 対策: リリースと同時にストア説明文を更新

### 変更後の Free/Pro 差分

| 差分項目 | Free | Pro |
|---------|------|-----|
| 写真枚数 | 10枚/レポート | 無制限 |
| PDF出力回数 | 月5回 | 無制限 |
| 透かし | 表紙に表示 | なし |
| 広告 | バナー表示 | なし |

## Tests

- Free版で「コメント」レイアウトを選択して PDF 出力できること
- 課金画面にレイアウト行が表示されないこと
- 設定画面の Pro 特典にレイアウト項目がないこと
- 既存 Pro モーダル（confirmLargeLayout）が表示されないこと
