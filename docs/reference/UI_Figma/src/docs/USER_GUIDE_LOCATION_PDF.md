# 📱 Repolog 使い方ガイド - 位置情報とPDF出力

## 🎯 このガイドについて

このガイドは、**小中学生でもわかるように**、Repologの新機能（位置情報とPDF出力）の使い方を説明します。

---

## 📍 位置情報を追加する方法

### STEP 1: レポートを作成する

1. ホーム画面の **「＋」ボタン** をタップ
2. **レポート名**を入力（例：「2025年1月26日 A現場」）
3. 写真を追加（カメラで撮影 or アルバムから選択）

### STEP 2: 位置情報を取得する

4. 画面を下にスクロール
5. **「位置情報」** のセクションを見つける
6. **「📍 位置情報を取得」** ボタンをタップ

### STEP 3: 位置情報の許可

7. ブラウザが「位置情報を使用しますか？」と聞いてくる
8. **「許可」** をタップ

> **ポイント：** 
> - 「拒否」を選ぶと、位置情報は取れません
> - 一度「拒否」しても、ブラウザの設定から後で許可できます

### STEP 4: 自動で取得される

9. ボタンが **「⏳ 取得中...」** に変わる
10. くるくる回るアイコンが表示される
11. 1〜3秒待つ

### STEP 5: 完了！

12. **「✓ 位置情報取得済み」** と緑色で表示される
13. 住所が表示される（例：「東京都渋谷区」）
14. 緯度経度も表示される（例：「Lat : 35.68945 , Lng : 139.70123」）

```
┌───────────────────────────────────┐
│ ✓ 位置情報取得済み                │
│ 東京都渋谷区                       │
│ Lat : 35.68945 , Lng : 139.70123  │
│ [🔄 再取得] [🗑 削除]              │
└───────────────────────────────────┘
```

---

## 🔄 住所を再取得する方法

位置情報は取れたけど、住所が「-」になっている場合：

1. **[🔄]** ボタンをタップ
2. 数秒待つ
3. 住所が表示される

> **なぜ「-」になるの？**
> - インターネットに繋がっていない
> - 電波が弱い
> - サービスが一時的にダウンしている

---

## 🗑 位置情報を削除する方法

間違った場所で取得してしまった場合：

1. **[🗑]** ボタンをタップ
2. 位置情報が削除される
3. もう一度 **「📍 位置情報を取得」** ボタンから取り直せる

---

## 📄 PDFを出力する方法

### STEP 1: レポートを完成させる

1. レポート名を入力
2. 写真を追加
3. コメントを入力
4. 天気を選択
5. 位置情報を取得（任意）

### STEP 2: PDFプレビューを開く

6. 右上の **「PDF」** ボタンをタップ
7. プレビュー画面が開く

### STEP 3: レイアウトを選ぶ

8. **「標準」** または **「大きく」** を選ぶ
   - **標準**：1ページに写真2枚
   - **大きく**：1ページに写真1枚（Pro版のみ）

### STEP 4: PDF出力

9. 下の **「PDF出力」** ボタンをタップ
10. プログレスバーが表示される

```
┌──────────────────────────────────┐
│ ⬇️ 読み込み中... 45%              │
└──────────────────────────────────┘
┌──────────────────────────────────┐
│ ████████████░░░░░░░░░░░░░░░░░░  │
└──────────────────────────────────┘
PDF生成中... 45%
```

11. 数秒〜数十秒待つ（写真の枚数による）
12. **完成！** PDFファイルが自動でダウンロードされる

### STEP 5: PDFを確認

13. ダウンロードフォルダを開く
14. ファイル名：`20250126_123045_現場A_Repolog.pdf`
15. PDFを開いて中身を確認

## ❓ よくある質問

### Q1: 位置情報が取れません

**原因と対策：**

#### ❌ 「位置情報の取得が拒否されました」
→ ブラウザの設定を確認してください

**Chrome の場合：**
1. アドレスバーの🔒アイコンをタップ
2. 「サイトの設定」をタップ
3. 「位置情報」を「許可」に変更

**Safari の場合：**
1. iPhoneの「設定」アプリを開く
2. 「Safari」をタップ
3. 「位置情報」を「許可」に変更

#### ❌ 「位置情報が利用できません」
→ GPS機能を有効にしてください

**iPhoneの場合：**
1. 「設定」→「プライバシーとセキュリティ」
2. 「位置情報サービス」をオンにする

**Androidの場合：**
1. 「設定」→「位置情報」
2. 位置情報をオンにする

#### ❌ 「位置情報の取得がタイムアウトしました」
→ 電波の良い場所で再試行してください

---

### Q2: 住所が「-」になります

**原因：**
- インターネットに繋がっていない
- 電波が弱い
- まれに、サービスがダウンしている

**対策：**
1. Wi-Fiまたは4G/5Gに接続
2. 電波の良い場所に移動
3. **[🔄 再取得]** ボタンをタップ

> **大丈夫！** 
> 住所が取れなくても、緯度経度は必ず表示されます。
> 緯度経度だけでも、Googleマップで場所を特定できます。

---

### Q3: PDFに住所が入りません

**チェック項目：**

✅ 位置情報を取得しましたか？
- 取得していない場合、PDFに位置情報は含まれません

✅ 住所は表示されていますか？
- 「-」の場合、PDFにも「-」と表示されます

✅ レポートを保存しましたか？
- 保存する前にPDFを出力すると、位置情報が含まれない場合があります

**解決方法：**
1. **「📍 位置情報を取得」** ボタンをタップ
2. 住所が表示されるのを待つ
3. **「保存」** ボタンをタップ
4. もう一度PDFを出力

---

### Q4: PDFが重い・遅い

**原因：**
- 写真の枚数が多すぎる
- 写真のサイズが大きすぎる

**推奨：**
- 📷 **写真は10枚以下**が快適
- 📷 **30枚以下**なら大丈夫
- 📷 **50枚以上**は時間がかかる（10秒以上）

**対策：**
1. 写真を減らす
2. 写真のサイズを小さくする（1920px幅以下推奨）
3. 複数のレポートに分割する

---

### Q5: アプリが固まった？

**症状：**
- PDFボタンを押したら、画面が動かない
- プログレスバーが表示されたまま

**大丈夫です！**
- 写真が多い場合、10〜30秒かかります
- プログレスバーを見て待ってください
- 45%で長時間止まることがありますが、処理中です

**本当に固まった場合：**
1. ブラウザをリロード（F5キー or 更新ボタン）
2. もう一度PDFを出力
3. それでもダメなら、写真を減らす

## 💡 便利な使い方

### 💡 現場でサクッと使う

1. 現場に到着したら、すぐ **「📍 位置情報を取得」**
2. 写真をパシャパシャ撮る
3. コメントをサクッと入力
4. その場で **「PDF出力」**
5. 元請けにメールで送信！

**所要時間：** 3〜5分

---

### 💡 一日の終わりにまとめる

1. 午前・午後で2つのレポートを作成
2. 各レポートに位置情報を追加
3. 写真を整理して追加
4. コメントを丁寧に入力
5. PDF出力して保存

**所要時間：** 10〜15分

---

### 💡 週報として使う

1. 月曜〜金曜の5日分のレポートを作成
2. 各レポートに位置情報を追加
3. 金曜の夕方にPDFを一括出力
4. 週報として提出

## 🎯 まとめ

### 位置情報の使い方

1. **「📍 位置情報を取得」** ボタンをタップ
2. ブラウザで「許可」を選択
3. 住所と緯度経度が自動で表示される
4. PDFに自動で含まれる

### PDFの使い方

1. レポートを完成させる
2. **「PDF」** ボタンをタップ
3. レイアウトを選ぶ
4. **「PDF出力」** をタップ
5. プログレスバーを見ながら待つ
6. 完成したPDFをダウンロード

---

## 📞 困ったら

### ブラウザを確認
- Chrome、Safari、Edge の最新版を使ってください
- Internet Explorer は使えません

### インターネット接続を確認
- Wi-Fiまたは4G/5Gに接続してください
- オフラインでは住所取得ができません

### 位置情報の設定を確認
- ブラウザで位置情報が「許可」になっているか
- スマホの設定でGPSが有効になっているか

### それでもダメなら
- ブラウザをリロード（更新）
- キャッシュをクリア
- ブラウザを再起動
- スマホを再起動
