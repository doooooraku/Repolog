## Design Principles

1. **Persona-first**: Each phrase uses words the target user (field/construction/maintenance worker) would naturally say
2. **Screen-matched**: Each phrase describes what the screenshot actually shows
3. **Not translated**: Each language was crafted independently for cultural authenticity
4. **ASO-aware**: Phrases include keywords users in each market would search for
5. **Length limits**: CJK ~15 chars, Latin ~35 chars (fits 1-2 lines on screenshot overlay)

## 実装上の注意

- エムダッシュ（—）の前後に thin space (`\u2009`) を入れない。Noto Sans で描画不可のため通常スペースまたはスペースなしとする
- Figma MCP でテキスト設定済み（19ページ）。Figma上のテキストが正とし、このファイルはリファレンス
- フォント: Noto Sans ファミリーで全19言語をカバー。Figma上ではInter Bold を使用
