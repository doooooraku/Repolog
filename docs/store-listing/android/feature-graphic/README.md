# Feature Graphic Specification

## Requirements (Google Play)
- Dimensions: 1024 x 500 px
- Format: PNG or JPEG (no alpha, no rounded corners)
- Used as the banner at the top of the store listing

## Design Guidelines
- Keep text minimal and readable at small sizes
- Include the app name "Repolog" prominently
- Show a visual representation of the core value: photos → PDF
- Use the app's brand colors
- Avoid placing critical content near edges (safe area: ~100px margin)

## Suggested Layout
```
+--------------------------------------------------+
|                                                    |
|    [Phone mockup showing      "Repolog"           |
|     Home screen with          Photos → PDF         |
|     report cards]             in seconds.          |
|                                                    |
+--------------------------------------------------+
```
